### 算法复杂度

#### 时间复杂度

```java
int sum(int n){
  int s = 0; //t
  for(;i<n;i++){  // t*n
    s = s+i // t*n
  }
  return s; // t
}
```

通过估算，代码执行时间 T(n)  与执行时间成正比

```
T(n) = O(f(n))

T(n): 代码执行时间
n: 数据规模
fn: 每行代码执行次数总和
O: 代码执行时间与 f(n) 表达式成正比
```



总结：上来就找 for 循环，1个循环就是 O(n) ， 2个嵌套就是 O(n^2)
