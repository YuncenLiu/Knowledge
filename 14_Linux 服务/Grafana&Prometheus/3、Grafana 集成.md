# Grafana 集成

官方下载地址：https://grafana.com/grafana/download

```sh
wget https://dl.grafana.com/enterprise/release/grafana-enterprise-10.1.2.linux-amd64.tar.gz
```

进入 bin 目录，启动

```sh
nohup ./grafana-server web > ./grafana.log 2>&1 &
```
